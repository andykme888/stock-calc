📈 股票做T计算器 (Stock T+0 Helper)
告别繁琐计算，让每一次“做T”都心中有数。

📱 应用简介
做T助手 Pro 是一款专为 A 股短线交易者和“做T”爱好者量身定制的安卓辅助工具。

在快节奏的股市博弈中，频繁的买卖操作往往让人难以记清真实的持仓成本和每笔操作的具体盈亏。本应用采用了专业的**“双轨制算法”，既能帮你计算不断降低的摊薄成本**，又能实时统计落袋为安的已实现盈亏。界面采用莫兰迪清爽配色，操作简单直观，数据本地自动保存，是您手机里的私人交易账本。

✨ 核心功能亮点
1. 🧠 独家“双轨制”核心算法
不同于普通软件简单的加减法，本应用同时维护两套数据：

摊薄成本 (做T视角)：通过低吸高抛，直观展示持仓成本的下降（甚至负成本），助你拿住牛股。

已实现盈亏 (会计视角)：每一笔卖出操作，系统都会立即结算该笔交易是赚是亏，并在仪表盘用红/绿颜色醒目显示累计收益。

2. 📊 全景数据仪表盘
首页顶部卡片式展示三大核心数据，一目了然：

📦 总持仓：当前手里的股票数量。

💰 摊薄成本：包含所有手续费后的实际保本价格。

🧧 已实现盈亏：实时显示“落袋为安”的真金白银（红涨绿跌）。

3. 📝 智能交易流水
专业报表：包含代码、名称、操作方向、均价、数量及详细分析。

趋势分析：每一笔操作后，系统会自动分析该操作是“加仓”还是“减仓”，并计算操作后的成本变化。

视觉增强：使用 Emoji 图标（🕹️、💲、📈）和颜色标签区分买卖，阅读体验极佳。

批量管理：支持勾选多条记录一键删除，容错率高。

4. 💾 数据自动记忆 (Auto-Save)
永不丢失：无论是买入、卖出、修改费率还是删除记录，系统都会毫秒级自动保存到手机本地。

开即是用：杀掉后台或重启手机后打开应用，所有数据瞬间恢复，无需重新录入。

5. ⚙️ 自定义费率系统
支持全自定义设置，满足不同券商标准：

佣金 (默认万2.5)

最低起步价 (默认 5元)

印花税 (卖出千0.5)

过户费 (万0.1)

🎨 界面设计
风格：采用现代化 Material Design 设计语言。

配色：护眼淡蓝灰背景 (#F0F2F5) + 纯白悬浮卡片，长期使用不累眼。

交互：大按钮设计，防止误触；输入框自动聚焦数字键盘，输入更丝滑。

🚀 适用人群
喜欢日内 T+0 操作的短线高手。

通过波段操作降低成本的价值投资者。

需要清晰记录每一笔交易费用的精细化管理者。

📥 既然有了它，为什么不用 Excel？
Excel 没法随身带：手机看盘时，切出来就能记，只需几秒钟。

Excel 很难算除权：本应用内置算法自动处理复杂的加减仓成本摊薄逻辑。

Excel 界面枯燥：这个应用好看，看着红红的盈亏数字，心情更好！
